# 指南

## 介绍

基于 React 移动端组件库。

### 特性

- 提供 70+ 套组件，覆盖移动端各类场景
- 支持主题定制
- 支持黑暗模式
- 支持国际化
- 支持 TypeScript

## 支持环境

- Android 5.0+
- iOS 10.0+
- Node.js 12.0+

## 框架版本

- react >= 16.9.0
- react-dom >= 16.9.0

## 安装

```sh
# npm
npm i -S @fox2/rfox

# yarn
yarn add @fox2/rfox

# pnpm
pnpm add @fox2/rfox
```
